# Проект: Pet Shelter Manager

## Описание:
Това е Java програма с графичен интерфейс за управление на животни в приют. Потребителят може да добавя, редактира и търси информация за животни и осиновители. Включени са динамични структури, сортиране, търсене и запазване на данни.

## Графична среда:
Eclipse IDE for Java Developers

## Автори:
Ясен Караджов

## JDK версия:
JDK 24.0.1 (изтеглена от jdk.java.net)

## Основни технологии:
- Java
- Swing
- Масиви, цикли, обектно-ориентирано програмиране
- Работа с .txt файлов
